# 食谱

每菜单一文件: `{菜名}.md`,YAML 头注明:

```yaml
---
name: 番茄龙利鱼
age: 12+月
prep_min: 10
cook_min: 15
allergens: [鱼]
source: 自创 | 出处链接 | 书名 + 页码
---
```

正文写: 食材 / 步骤 / 给钦钦的反馈记录(吃了多少、是否爱)。

源头建议:
- AAP HealthyChildren 一日食谱样例(`feeding_toddler.md` 引用源里有)
- 中国营养学会《中国 7-24 月龄婴幼儿喂养指南》
- 自创不引用即可,但标 `source: 自创`
